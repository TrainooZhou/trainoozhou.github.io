ELK安装以及配置

ELK是Elasticsearch、Logstash、Kibana的简称，这三者是核心套件，但并非全部。

关于这三者的具体介绍可以移步官方网站查看详情，此文介绍如何安装ELK

一、部署环境

  • CentOS-6.x
  • Java-1.8
  • ELK-7.4.0
